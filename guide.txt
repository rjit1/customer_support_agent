### ğŸ§  2. Supabase Database Schema

```sql
-- Users Table
CREATE TABLE users (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  email TEXT,
  name TEXT,
  preferred_language TEXT,
  created_at TIMESTAMP DEFAULT now()
);

-- Chats Table
CREATE TABLE chats (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id uuid REFERENCES users(id),
  message TEXT,
  role TEXT, -- 'user' or 'assistant'
  created_at TIMESTAMP DEFAULT now()
);
```
